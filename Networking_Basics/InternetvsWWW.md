## The Internet is a global network of interconnected computers and devices that communicate using the TCP/IP protocol suite. It provides the infrastructure for services like websites, email, file transfer, online gaming, and video conferencing. 

## The World Wide Web, or WWW, is one of those services. It is a collection of interconnected web pages and resources that users access through a web browser using HTTP or HTTPS. 

### In short, the Internet is the network, while the Web is a service that runs on top of that network.

# Note : Internet and WWW are not same
Internet = Infrastructure (network)
WWW = A service running on top of the Internet
